Сценарии для тестирования функционала веб-сайта онлайн-магазина.

1. Регистрация пользователя:
  - Проверка наличия всех обязательных полей в форме регистрации
  - Проверка уникальности выбранного имени пользователя и электронной почты
  - Проверка возможности входа после регистрации
  - Проверка подтверждения регистрации через электронную почту

2. Добавление товара в корзину:
  - Проверка наличия кнопки добавления в корзину на странице товара
  - Проверка изменения количества товаров в корзине после добавления
  - Проверка возможности удаления товара из корзины
  - Проверка общей суммы заказа и количества товаров в корзине

3.Поиск товара:
  - Проверка наличия поля для ввода поискового запроса
  - Проверка корректности отображения результатов поиска
  - Проверка корректности сортировки результатов поиска
  - Проверка корректности работы фильтров поиска

4. Оформление заказа:
  - Проверка наличия всех необходимых полей в форме заказа
  - Проверка корректности подсчета итоговой стоимости заказа
  - Проверка корректности выбора способа доставки и оплаты
  - Проверка возможности отмены заказа до его подтверждения

5. Работа сайта на разных устройствах:
  - Проверка корректности отображения сайта на мобильных устройствах и планшетах
  - Проверка корректности отображения сайта на разных браузерах и операционных системах
  - Проверка корректности работы всех функций сайта на всех устройствах
  - Проверка корректности отображения сайта в разных режимах экрана (например, в вертикальном и горизонтальном положении)
